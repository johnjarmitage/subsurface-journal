We will use Sphinx which is a tool that makes it easy to create intelligent and beautiful documentation.

The documentation will be automatically generated whenever a modification is committed to the repository. The documentation is hosted on GitLab Pages and it is beautifully formatted using the ReadTheDocs theme. This blog post also gives a nice overview of Sphinx and ReadTheDocs.

We use the ReStructuredText format to write our documentation. This blog post explains why ReStructuredText is better than Markdown for writing documentation.
